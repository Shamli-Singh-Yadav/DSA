##what is an array?
A container that stores multiple values of the same type in one place, under one name.The size of an array is static(they cannot be chaned after declaration). We can update and change the values of the array elements.

###Creation of arrays:
dataType arrayName[]=new dataType[size];
Examples:
1. int arr[]=new int[5];
2. string fruits[]={"apple","banana","grape"};

###Input/output in array:
int marks[]=new int[5];
Scanner sc=new Scanner(System.in);
marks[0]=sc.nextInt();//input
System.out.println(marks[0]);//output

###Function we can perform on arrays:

1. Traversing: accessing each element of the array one by one.
2. Insertion: adding an element at a particular index.
3. Deletion: removing an element from a particular index.
4. Searching: finding the index of a particular element.
5. Sorting: arranging the elements in a particular order(ascending or descending).
6. To find the length of the array: use arrayName.length property.
